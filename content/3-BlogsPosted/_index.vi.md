Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Ví dụ:

###  [Blog 1 - VPC ENDPOINT — CHÌA KHÓA ĐỂ XÂY SANDBOX CÔ LẬP TRÊN AWS](3.1-Blog1/)
Bài viết chia sẻ cách xây dựng mạng Sandbox cô lập hoàn toàn cho hệ thống phân tích mã độc trên AWS bằng cách dùng VPC Endpoints (S3, DynamoDB, CloudWatch Logs) mà không cần dùng Internet Gateway hay NAT Gateway.

###  [Blog 2 - S3 PRESIGNED URL — UPLOAD FILE AN TOÀN MÀ KHÔNG CẦN ĐI QUA SERVER](3.2-Blog2/)
Bài viết phân tích giải pháp dùng S3 Presigned URL để client upload trực tiếp file lên S3, giúp giải quyết triệt để giới hạn payload của API Gateway và Lambda, đồng thời tối ưu chi phí vận hành.

###  [Blog 3 - AWS CONFIG AUTO-REMEDIATION — TỰ ĐỘNG KHÓA S3 BUCKET TRONG 1 GIÂY KHI AI ĐÓ LỠ MỞ PUBLIC](3.3-Blog3/)
Bài viết giới thiệu giải pháp tự động khắc phục cấu hình bảo mật sử dụng bộ ba dịch vụ AWS Config, EventBridge và Lambda để tự động chặn Public Access của S3 Bucket ngay lập tức khi phát hiện vi phạm.
